Для начала устанавливаем последнию версию Python, скачать её можно тут
https://www.python.org/ftp/python/3.9.6/python-3.9.6-amd64.exe 
При установке не забываем нажать на кнопку "Add Python 3.9 to PATH"

После установки открываем командную строку (cmd), и прописываем туда следующее
pip install PyTelegramBotApi
pip install SimpleQIWI

После чего открываем файл config.py и настраиваем его под себя. Каждую переменную я подписал.
Открываешь файл main.py, ищешь 56 строку, и меняешь путь к созданию бд в папку с ботом.
Например: "C:/team/bot/database.db"
Готово, открываем main.py и проверяем работоспособность бота.
Через БД выдаем себе админку

Держать бота нужно на выделенном сервере. Или же VDS или Дедик. 
Как пользоваться дедиками/VDS, написанно в интернете. Умейте пользоваться гуглом.
